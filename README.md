# BagiCerita
BagiCerita adalah ruang berbagi dan berdiskusi bagi mahasiswa untuk saling mendengarkan pengalaman, perjalanan, dan pandangan hidup. Setiap Selasa malam, satu mahasiswa akan berbagi kisah tentang magang, keahlian, atau pengalaman pribadi yang bisa menginspirasi. Kegiatan ini diadakan secara online melalui Google Meet dan terbuka untuk semua.
