Ini adalah kode sumber (source code) untuk website komunitas **BagiCerita**.

BagiCerita adalah ruang berbagi dan berdiskusi bagi mahasiswa untuk saling mendengarkan pengalaman, perjalanan, dan pandangan hidup. Website ini berfungsi sebagai landing page untuk menginformasikan jadwal, format kegiatan, tujuan komunitas, serta cara bergabung.

**Tagline:** "Berani Bicara, Mau Mendengar."

## ğŸš€ Fitur Utama Website

Website ini dibangun sebagai *Single Page Application (SPA)* statis dengan fokus pada kecepatan dan kemudahan akses.

* **Responsif Penuh:** Tampilan optimal di perangkat seluler maupun desktop.
* **Minimalis & Informatif:** Menyajikan semua detail komunitas (Tujuan, Format Kegiatan, Visi) dalam satu halaman.
* **Branding Kuat:** Menggunakan palet warna Biru Tua dan Oranye yang konsisten dengan logo.
* **Call-to-Action (CTA):** Tautan langsung ke formulir pendaftaran narasumber dan informasi jadwal.
* **Teknologi Ringan:** Menggunakan HTML murni dan Tailwind CSS via CDN.
